一键安装LAMP服务：

	sudo tasksel install lamp-server

一键卸载LAMP：

	sudo tasksel remove lamp-server

补充：

	LAMP = Linux+Apache+MySQL+PHP

通过上面的命令卸载Lamp时不免把Linux系统本身的东西卸载掉了，因此，

在卸载LAMP后**一定**记着更新一下系统：

	sudo apt-get update
	sudo apt-get upgrade

上面两条都要执行，切记！
